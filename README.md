# 新聞神曲製片工具

新聞網址 → 洗腦 Rap 歌詞 → 音樂 + 畫面素材 → 剪映組裝

## 環境變數（Railway 設定）

| 變數名稱 | 說明 |
|---------|------|
| `GEMINI_API_KEY` | Google Gemini API 金鑰 |
| `KIE_API_KEY` | kie.ai API 金鑰 |
| `SITE_PASSWORD` | 網站存取密碼 |

## 部署步驟

1. 把這個資料夾推到 GitHub
2. 登入 railway.app → New Project → Deploy from GitHub
3. 設定三個環境變數
4. 部署完成，取得公開網址

## 每則影片成本

| 項目 | 點數 | USD |
|------|------|-----|
| Suno 音樂 × 2版本 | 24 | $0.12 |
| grok-imagine 畫面 × 3片段 | ~29 | $0.14 |
| Gemini 分析 | - | ~$0.01 |
| **合計** | **~53** | **~$0.27** |

## 使用流程

1. 貼上新聞網址 → AI 自動分析生成歌詞
2. 編輯歌詞（可修改每一句）
3. 確認 Suno 音樂風格指令 → 生成兩個版本音樂
4. 選一個音樂版本
5. 生成 3 個關鍵畫面片段
6. 下載素材 → 進剪映組裝
